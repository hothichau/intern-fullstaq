== Description ==

This plugin adds a cookie notification to your website. It blocks cookies until the user accepts them.
Works with Multi sites and Multi language (WPML) and it has some styling options for the notification.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/eu-cookie-manager` directory
2. Activate the plugin through the 'EU Cookie Manager' menu in WordPress
